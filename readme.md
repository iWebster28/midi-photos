# midi-photos

## To install:
- Create a virtual environment  
```python3 -v venv env```
- Activate virtual environment (macOS)  
```source env/bin/activate```
- Install requirements  
```pip3 install -r requirements.txt```

## To run:
- Connect Samson Graphite MF8 to your computer  
- Run
```python3 main.py```
